# Code Hotspot Analysis - Sample Repo List

# Format: one repo path per line
# Optional per-repo patterns: include:*.java exclude:build/*
# Per-repo patterns override global --include/--exclude flags
# Lines starting with # are comments
#
# Examples:
#   /path/to/repo                                         — no per-repo filters (uses global)
#   /path/to/java-repo include:*.java exclude:build/*     — Java repo, exclude generated code
#   /path/to/js-repo include:*.js include:*.ts            — JS/TS only, include all
